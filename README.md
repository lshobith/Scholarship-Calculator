# Scholarship-Calculator

## requirements

###### Django 1.11
```bash
$ pip install django
```

###### oauth2client
```bash
$ pip install --upgrade oauth2client
```

## setup commands
```bash
$ python manage.py makemigrations app
$ python manage.py migrate
$ python manage.py collectstatic
```

A website to find scholarships.
